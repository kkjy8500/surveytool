# Survey Tool — 조사설정 엑셀 기반 개편본

## 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 작업 흐름

1. CSV/XLSX/SAV 데이터 파일 업로드
2. 표준 조사설정 XLSX 업로드
3. 통계표 엑셀 추출
4. 그래프 PNG 또는 대시보드 패키지 생성

콜럼가이드가 없는 경우 앱의 `설문지에서 조사설정 초안 만들기` 기능을 사용합니다.

## 삭제된 기능

- 데이터 검수
- 조사 설정 임시저장
- 재사용 스냅샷/템플릿
- 웹 화면에서의 문항유형·배너 수동 편집
- 리코딩 및 복잡한 단계별 세션 상태

## 조사설정 기본값

- 비율 기준: 유효응답
- 전체 행 포함
- N 및 비율 출력
- 척도형: 평균·표준편차 추가
- 복수응답: 값 1을 선택으로 처리
- 가중치: 데이터에 `wt`, `weight`, `가중치`, `표본가중치`가 있으면 자동 적용
